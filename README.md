# EV FastAPI Project 🚗⚡

A machine learning project using **Random Forest** to model and predict Electric Vehicle (EV) data, deployed with **FastAPI** and containerized using **Docker**.

---

## 📖 Features
- Data preprocessing and model training with scikit-learn
- Random Forest model for EV prediction
- REST API built with FastAPI
- Dockerized for easy deployment
